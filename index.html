📚 Dokumentasi Takon AI

🤖 Apa itu Takon AI?

Takon AI adalah asisten virtual berbasis kecerdasan buatan yang dapat menjawab berbagai pertanyaan, membuat konten, menerjemahkan, menulis kode, dan banyak lagi. Dibangun dengan teknologi model bahasa besar (LLM) yang diakses melalui Groq API.

🚀 Cara Menggunakan

  • Ketik pertanyaan atau perintah di kotak input pada halaman Chat.
  • Gunakan Enter untuk mengirim, dan Shift+Enter untuk membuat baris baru.
  • Kamu juga bisa mengklik chip saran untuk memulai percakapan.
  • AI akan merespons secara real-time dengan streaming teks.

✨ Fitur Unggulan

  • Chat Streaming: Jawaban muncul kata demi kata.
  • Multi-bahasa: Meskipun fokus bahasa Indonesia, AI bisa memahami berbagai bahasa.
  • Kode & Markdown: Balasan mendukung format teks kaya, blok kode dengan tombol salin.
  • Mode Gelap: Klik ikon bulan/matahari untuk beralih tema.
  • Profil Pengguna: Sesuaikan nama dan bio kamu di halaman Profil.
  • Keamanan: API key tidak pernah dikirim ke browser, semua komunikasi via server Vercel.

🔧 Teknologi

Frontend murni HTML/CSS/JS dengan desain glassmorphism. Backend menggunakan Vercel Edge Function yang meneruskan permintaan ke Groq API. Model default: llama-3.1-8b-instant.
